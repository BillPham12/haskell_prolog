The program take an input from an user then it prints out a list of
 "Pig Latin" strings instead of the original input.

Warning:
If the input has any capital letters, the program will print out the warning message instead.

To compile the program:
1) javac comp3007_w19_101023249_a1_1.java
2) java comp3007_w19_101023249_a1_1

If you wanna test the program without using scanner, you can change the input value in line 19
with your string input such as "my_name_is_your_name".
